## Prompt Genérico
Pergunta: O que é Machine Learning?

Resposta:
Machine Learning é uma área da inteligência artificial que permite que computadores aprendam a partir de dados, sem serem programados diretamente para cada tarefa. Ele usa algoritmos para identificar padrões e tomar decisões com base nesses padrões.

---

## Prompt Estruturado
Prompt:
Atue como um professor. Explique Machine Learning para um iniciante usando uma analogia com esportes em 2 parágrafos.

Resposta:
Machine Learning pode ser comparado ao treinamento de um atleta. Assim como um jogador de futebol melhora suas habilidades com prática e repetição, os computadores também aprendem analisando muitos dados. Quanto mais dados recebem, melhor ficam em reconhecer padrões.

Por exemplo, um goleiro aprende a prever a direção da bola observando vários chutes. Da mesma forma, um sistema de Machine Learning analisa exemplos para prever resultados futuros, como identificar se um e-mail é spam.
